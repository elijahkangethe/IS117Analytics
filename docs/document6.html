How fingerprinting browsers is used for advretising

The Biggest threat to Privacy: Browser Fingerprints

Website fingerprints and data

Browser Fingerprinting is a web tracking technique that is much more sophisticated and invasive than basic internet browser cookies. Browser Fingerprinting is when a user browses the internet and advertising networks recognize the computer based on it’s hardware specifications, type of device and other factors to create an advertising profile for that specific computer and the advertisers can maintain that data and track the user throughout the internet. Moreover, what enables browser finger printing is the use of JavaScript in web pages as stated in the ScottiesTechInfo video .”So using javascript I can find out all kinds of interesting bits of information such as I can know what web browser you’re using, I can know what operating system you’re using, I can find out the device type.”(2:40-3:20) Therefore, specific information is effective for advertising agencies to know the exact specifics about how a person browses the internet about the device they are using. Moreover, advertisers can use the device information about the user to send targeted ads depending on which device they are using. For example, an advertiser can send ads to a person’s phone for phone cases and phone chargers. However, if someone is using a desktop computer they can send ads for computer accessories such as keyboards, mice, and other computer items.
